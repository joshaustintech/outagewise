# OutageWise

OutageWise is an HTTP outage detection service with customer-visible status
pages, analytics, and an authenticated read-only MCP server for realtime outage
visibility.

## Product Planning

The product roadmap is maintained in Markdown so any coding agent can use it
without vendor-specific assumptions:

- [Product roadmap](docs/product-roadmap.md)
- [Demoable milestones](docs/milestones.md)
- [Agent task breakdown](docs/agent-task-breakdown.md)

Each milestone is designed to be manually demoable and testable. Each task is
small enough to hand to a compact coding model with the context it needs.
